HTML Flashcards

<a>

The Anchor Tag

The anchor element is used with an href attribute to create a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Example:

<a href="https://www.example.com">Website</a>

would appear as

Website

<abbr>

The Abbreviation Tag

The abbreviation element is used with a title attribute to create an abbreviation or acronym with a description.

Example:

<abbr title="HyperText Markup Language">HTML</abbr>

would appear as

HTML

<button>

The Button Tag

The button element represents a clickable button and is usually used to submit forms; however, its behavior can be altered with the type attribute.

Example:

<button type="button">Do Nothing</button>

would appear as

<code>

The Inline Code Tag

The code element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.

Example:

<code>let var = true;</code>

would appear as

let var = true;

<datalist>

The Data List Tag

The datalist element contains a set of option elements that represent the recommended options available to choose from within other controls. Datalist elements are used alongside input elements and they connect with the id and list attributes.

Example:

<label for="ice-cream-choice">Choose a flavor:</label>
<input list="ice-cream-flavors" id="ice-cream-choice">
<datalist id="ice-cream-flavors">
    <option value="Chocolate">
    <option value="Mint">
    <option value="Strawberry">
    <option value="Vanilla">
</datalist>

would appear as

<details>

The Details Disclosure Tag

The details element creates a disclosure widget in which information is visible only when the widget is toggled into an open state. A label must be provided using the summary element.

Example:

<details>
    <summary>Secret Message</summary>
    You found me!
</details>

would appear as

Secret Message

You found me!

<em>

The Emphasis Tag

The emphasis element marks text that has stress emphasis. It can be nested, with each level of nesting indicating a greater degree of emphasis. Though the emphasis element applys italic type, that is not its primary purpose and it should not be used just for styling.

Example:

<p>Do <em>not</em> use emphasis for italic styling.</p>

would appear as

Do not use emphasis for italic styling.

<fieldset>

The Field Set Tag

The fieldset element is used to group several controls, such as inputs, as well as labels within a web form. It can have a nested legend element to provide a caption.

Example:

<fieldset>
    <legend>Choose your favorite monster</legend>
    <input type="radio" id="godzilla" name="monster">
    <label for="godzilla">Godzilla</label>
    <input type="radio" id="mothra" name="monster">
    <label for="mothra">Mothra</label>
    <input type="radio" id="ghidorah" name="monster">
    <label for="ghidorah">Ghidorah</label>
    <input type="radio" id="rodan" name="monster">
    <label for="rodan">Rodan</label>
</fieldset>

would appear as

Choose your favorite monster

<select>

The Select Tag

The select element represents a control that provides a menu of options. It functions like a dropdown menu that allows the user to select one option.

Example:

<label for="pet-select">Choose a pet:</label>
<select name="pets" id="pet-select">
    <option value="">--Please choose an option--</option>
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
    <option value="hamster">Hamster</option>
</select>

would appear as

<strong>

The Strong Importance Tag

The strong element indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type, but it should not be used to apply bold styling.

Example:

<p>Use the strong element for importance <strong>not for styling</strong>.</p>

would appear as

Use the strong element for importance not for styling.

<sub>

The Subscript Tag

The subscript element specifies inline text which should be displayed as subscript for solely typographical reasons. Subscripts are typically rendered with a lowered baseline using smaller text.

Example:

<p>Everyone loves H<sub>2</sub>O.</p>

would appear as

Everyone loves H2O.

<sup>

The Superscript Tag

The superscript element specifies inline text which is to be displayed as superscript for solely typographical reasons. Superscripts are usually rendered with a raised baseline using smaller text.

Example:

<p>2<sup>3</sup> = 8</p>

would appear as

23 = 8
